# Escape the shelter


### Escape the shelter - игра "выберись из лабиринта" стилизована под старые игры PlayStation 1.            


#### Оригинально является лабораторным проектом по предмету Структуры и Алгоритмы Обработки данных.

> Благодарность Брюсу Бучеру за идею геймдизайна.                                         
> Использовался текстур-пак Modern Interiors от LimeZu.

Управление:
- Мышь - движение персонажа.
- Левая кнопка мыши - действие.

Если выбранный предмет интерактивен - около курсора появится знак действия ("?").
